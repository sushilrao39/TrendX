# 🛍️ TrendX – MERN Stack E-Commerce Website

TrendX is a full-stack e-commerce web application built using the MERN stack (MongoDB, Express, React, Node.js).
It allows users to browse products, add items to cart, and place orders with a modern responsive UI.

---

## 🚀 Features

* 🔐 User Authentication (Login / Register)
* 🛒 Add to Cart functionality
* 📦 Product listing and filtering
* 🧑‍💼 Admin panel for product management
* 💳 Order management system
* 📱 Fully responsive design
* ⚡ Fast React frontend

---

## 🧑‍💻 Tech Stack

**Frontend**

* React.js
* Tailwind CSS
* Axios

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

## ⚙️ Installation

### 1️⃣ Clone the repository

git clone https://github.com/sushilrao39/Mern-stack-E-commerce.git

### 2️⃣ Go to project folder

cd Mern-stack-E-commerce

### 3️⃣ Install dependencies

Backend

cd backend
npm install

Frontend

cd frontend
npm install

### 4️⃣ Start the application

Backend

npm run server

Frontend

npm run dev
